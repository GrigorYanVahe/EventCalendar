Тестовое iOS-приложение с календарём и событиями, сфокусированное на архитектуре и работе с данными.

Основное: 
	•	SwiftUI
	•	MVVM + Coordinator
	•	Современный UI с поддержкой Light / Dark Mode
	•	Собственные модели данных для календаря и событий

Данные: 
	Данные разделены на:
	•	информацию о днях месяца (наличие событий)
	•	список событий выбранного дня
	•   JSON-файлы сгенерированы с помощью AI
	•   Обработка данных выполняется на стороне приложения для симуляции API (фильтрация по месяцам и дням). В реальных проектах такая логика обычно находится на сервере.

Загрузка данных: 
	Используется `URLSession` как для локальных данных, так и для удалённых (remote).
  По умолчанию данные загружаются удалённо из GitHub через `URLSession`.
	Тип загрузки данных можно переключать в `EventCalendar/Services/NetworkService`.

Состояние загрузки: 
	•	Есть индикация загрузки событий дня
	•	Может отображаться очень короткое время из-за малого объёма данных

Требования: 
	•	iOS 17+
	•	Xcode 15+
	•	Swift 5.9+

  ## Запуск проекта

1. Клонировать репозиторий.
2. Открыть `EventCalendar.xcodeproj`.
3. Выбрать iOS Simulator или реальное устройство.
4. Запустить проект (`Cmd + R`).
